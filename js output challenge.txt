
### Classic Closure Problem (var)

for (var i=0;i<3;i++) {
setTimeout(() =>console.log(i),0)
}

🔹 What is the output? 3 , 3 , 3  ? why because the callback function work after the loop end and we only have one i in the program and the callback function save the reference of the i not its value 
# but with let creates a brand new i for every single iteration of the loop, Each callback gets its own private copy of i frozen at that iteration's value.
